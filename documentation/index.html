Adaptagrams project documentation

Adaptagrams is a library of tools and reusable code for adaptive diagramming applications, for example: drawing tools, automated document and diagram layout, smart presentation software, graph drawing, chart layout, etc.

The Adaptagrams project includes several cross-platform C++ libraries. You find find their individual documentation via the links below:

  • libvpsc — a solver for the Variable Placement with Separation Constraints problem.
  • libcola — a library for force-directed network layout using the stress-majorization method subject to separation constraints.
  • libavoid — a library providing object-avoiding polyline and orthogonal connector routing for use in interactive diagram editors.
  • libtopology — extensions to other libraries to support topology-preserving layout.

These libraries are collectively known as COLA (for COnstraint LAyout).

All code in the Adaptagrams repository is released as open source under the terms of the LGPL.

We also dual-license the Adaptagrams libraries and for a fee we can provide them under a less-restrictive commercial license as well as extend them to fit your needs (contact us). For this reason, if you contribute code to the project and would like it to appear in the main Adaptagrams repository, we require that you assign the copyright on your changes to Monash Univeristy with the following statement: "I hereby assign copyright in this code to Monash University, to be licensed under the same terms as the rest of the code."

The Adaptagrams code is written and maintained by Tim Dwyer and Michael Wybrow, members of the MArVL: Monash Adaptive Visualisation Lab at Monash University. A number of papers describing the algorithms used in these libraries are mentioned on the documentation pages for the individual libraries.
